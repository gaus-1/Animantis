# 🧠 Платформа AI-Агентов для РФ
## Улучшенный MoltBot, масштабированный на миллионы

---

## Что такое MoltBot и как он работает

```
Пользователь → [Telegram/WhatsApp/Discord] → Gateway (Node.js) → LLM (Claude/GPT) → Действия (файлы, браузер, API)
                                                    ↕
                                              Память (Markdown)
                                                    ↕
                                              Скиллы (плагины)
```

**Проблемы MoltBot** (реальные жалобы пользователей Reddit):

| Проблема | Детали |
|---|---|
| 💸 **$300–750/мес** | «Burns tokens like a jet engine» — 8M токенов на дорогих моделях |
| 🔒 **Кошмар безопасности** | Полный доступ к ОС, нет слоя разрешений, prompt injection |
| 🛠 **Нестабильная установка** | «Very vibe-coded», плохая документация, API-ошибки |
| 🤷 **Только для гиков** | Self-hosted = нужен сервер, Node.js, CLI |
| 👤 **Один юзер = один бот** | Нет многопользовательской архитектуры |
| 🚫 **Нет социального слоя** | Каждый юзер изолирован, нет обмена скиллами и автоматизациями |

## Наша платформа: что делаем по-другому

### Ключевая идея

> **MoltBot = личный инструмент для гиков.**
> **Мы = социальная платформа AI-агентов для всех.**

## 6 ключевых улучшений над MoltBot

### 1. ☁️ Облачный, не self-hosted
**MoltBot:** нужен сервер, CLI, Node.js, Docker  
**Мы:** зарегистрировался в Telegram → написал боту «Привет» → готов к работе  

> Зайти в Telegram может каждый. Поднять VPS с Node.js — нет.

### 2. 💰 В 10-50x дешевле
**MoltBot:** $300–750/мес (Claude 4.5 Opus, 8M токенов/день)  
**Мы:**
- YandexGPT Foundation — дешевле западных моделей
- Умный роутинг: простые задачи → дешёвая модель, сложные → мощная
- Кэширование ответов на типовые запросы
- **Цель: 990₽/мес ($10)** за Pro-тариф

### 3. 🔒 Безопасность из коробки
**MoltBot:** «security nightmare», полный доступ к ОС  
**Мы:**
- Sandboxed-исполнение (агент не трогает ОС юзера напрямую)
- Явные разрешения: «Агент хочет прочитать файл X — разрешить?»
- Аудит-лог всех действий
- Для бизнеса: изоляция данных, шифрование, SOC2-ready

### 4. 🌐 Социальный слой (главное отличие)

| Фича | Что делает | Аналог |
|---|---|---|
| **Маркетплейс скиллов** | Покупай/продавай автоматизации | App Store |
| **Лента автоматизаций** | «Посмотри, что делает агент Васи» | TikTok/Instagram |
| **Клонирование агентов** | «Мне нравится твой агент — хочу такого же» | Fork на GitHub |
| **Рейтинги и отзывы** | Лучшие скиллы поднимаются вверх | Product Hunt |
| **Команды** | Несколько человек → один агент | Slack workspace |

**Почему это создаёт сетевой эффект:**
- Чем больше людей → тем больше скиллов в маркетплейсе
- Чем больше скиллов → тем полезнее платформа
- Чем полезнее платформа → тем больше людей
- **= Flywheel, который раскручивается сам**

### 5. 🤖 Мульти-агентная архитектура
**MoltBot:** один бот = один агент  
**Мы:** У каждого юзера может быть **команда агентов**:

| Агент | Специализация | Пример |
|---|---|---|
| 📊 Аналитик | Анализ данных, отчёты | Ежедневная сводка продаж из WB |
| ✍️ Контент-мейкер | Тексты, SEO, соцсети | Автопостинг в VK и Telegram |
| 📞 Помощник | Календарь, напоминания | «Через час встреча с клиентом» |
| 🛒 Закупщик | Мониторинг цен, заказы | «Цена на товар X упала на 20%» |

Агенты **общаются между собой**: Аналитик видит тренд → сообщает Контент-мейкеру → тот готовит пост.

## Техстек

| Компонент | Технология | Почему |
|---|---|---|
| **Backend** | Python + FastAPI | Ты это знаешь, экосистема AI |
| **Agent Runtime** | LangGraph / собственный | Мульти-агентная оркестрация |
| **LLM** | YandexGPT + GigaChat + Llama | Дёшево, суверенно |
| **База данных** | PostgreSQL + pgvector | Ты это настраивал в PandaPal |
| **Кэш/Очереди** | Redis | Ты это настраивал в PandaPal |
| **Messaging** | Telegram Bot API + Webhooks | Ты это делал в PandaPal |
| **Frontend** | Telegram Mini App (React) | Ты это делал в PandaPal |
| **Плагины** | MCP Protocol | Стандарт индустрии |
| **Деплой** | Docker + VPS (Timeweb/Selectel) | Российские провайдеры |
