## Что такое MCP-сервер (за 30 секунд)

```
AI (Claude/GPT/YandexGPT)  ←→  MCP-сервер  ←→  Внешний сервис (1С, WB, Ozon...)
                                    │
                           «Переводчик» между
                            AI и реальным миром
```

MCP-сервер = **плагин для AI**. Он даёт AI доступ к сервисам, которых AI сам не знает. Написал MCP-сервер для Wildberries → любой AI может управлять магазином на WB.

**Рынок сейчас:** 18 473 MCP-сервера на mcp.so. 33 категории. Растёт лавинообразно.

## Как делать (технически)

### Стек: Python + FastMCP

```python
# Это ВЕСЬ код минимального MCP-сервера:
from fastmcp import FastMCP

mcp = FastMCP("wildberries-server")

@mcp.tool()
def get_orders(api_key: str, date_from: str) -> dict:
    """Получить заказы из Wildberries за период"""
    import requests
    response = requests.get(
        "https://statistics-api.wildberries.ru/api/v1/supplier/orders",
        headers={"Authorization": api_key},
        params={"dateFrom": date_from}
    )
    return response.json()

@mcp.tool()
def reply_to_feedback(api_key: str, feedback_id: str, text: str) -> dict:

@mcp.resource("wb://daily-report")
def daily_report() -> str:

mcp.run()
```

### 6 шагов создания MCP-сервера

| Шаг | Что делать | Время |
|---|---|---|
| 1. **Изучить API** | Прочитать документацию целевого сервиса | 2–4 часа |
| 2. **Определить tools** | Какие действия нужны (заказы, отзывы, цены) | 1 час |
| 3. **Написать код** | Python + FastMCP, обработка ошибок | 4–8 часов |
| 4. **Тесты** | MCP Inspector / Claude Desktop | 2–3 часа |
| 5. **Документация** | README, описания tools на русском/английском | 2 часа |
| 6. **Публикация** | mcp.so, Smithery, npm/PyPI, GitHub | 1 час |

**Итого: 1–2 дня на один MCP-сервер.**

### Модель 2: Pay-per-call (через xpay)

Платформа xpay позволяет поставить платёжный слой перед MCP-сервером **без единой строки кода:**

```
AI вызывает tool → xpay списывает $0.01 → MCP-сервер выполняет → результат
```

- WB: получить заказы = $0.005 за вызов
- WB: ответить на отзыв = $0.02 за вызов
- 1С: создать документ = $0.05 за вызов

Публикуешь на mcp.so / Apify → платформа берёт 20-30% → ты получаешь 70-80%.

## Почему это продаётся само

1. **Поиск решает** — продавец WB гуглит «MCP Wildberries» → находит тебя → ты единственный
2. **npm/PyPI = дистрибуция** — `pip install mcp-wildberries` находится автоматически
3. **mcp.so = витрина** — 18 000+ серверов, но 0 для российских сервисов
4. **Статьи = магнит** — «Как AI управляет моим WB» на vc.ru = тысячи просмотров
5. **Чаты продавцов = вирус** — один показал → десять поставили

> [!TIP]
> **Начинай с Wildberries MCP.** Продавцов WB — 500 000+, боль реальная, API открыт, конкурентов ноль. Один MCP-сервер за 2 дня → первые юзеры за неделю.
